# student-result management-system

This Python application manages student records by utilizing Object-Oriented Programming (OOP) principles. Users can input information for multiple students, while the system automatically calculates grades and determines pass or fail statuses based on scores. Additionally, the project implements file handling to securely store and preserve all student data. 

* FEATURES :
1) Capability to input multiple student profiles simultaneously.
2) Maintain essential information including student name, unique roll number, and academic scores.
3) Instantly computes student grades based on performance.
4) Automatically determines and outputs whether a student has passed or failed.
5) Exports and saves compiled student information directly to a local file.
6) Accesses, loads, and neatly presents the stored student records.
7) Structured around core OOP principles, utilizing classes, objects, and constructors.

* CONCEPTS USED :
1)Python
2)OOPs(Constructor, class, Object)
3)Lists
4)Conditional statements
5)File Handling(Read, Write, Append)
